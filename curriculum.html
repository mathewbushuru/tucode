Take a look at our curriculum ...

We will be using the Raspberry Pi Digital Making Curriculum. The curriculum has four levels

1. Creator
- Can use basic programming constructs to create simple programs.

2. Builder
- Can apply basic programming constructs to solve a problem.

3. Developer
- Can apply abstraction and decomposition to solve more complex problems.

4. Maker
- Can apply higher order programming techniques to solve real world problems.
